# Tlapse

Automated your time lapse photography using gphoto2.

## Installation

1. Install gphoto2

  * OSX

    ```
    brew install gphoto2
    ```

  * Linux

    ```
    sudo apt-get install gphoto2
    ```

2. `gem install tlapse`
